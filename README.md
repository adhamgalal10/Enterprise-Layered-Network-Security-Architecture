# Enterprise-Layered-Network-Security-Architecture
Designed and deployed a secure, defense-in-depth enterprise network topology utilizing GNS3 to segment trusted corporate domains from untrusted boundaries.  Configured a stateful edge gateway with routing and iptables firewalling to drop unauthorized external payloads, achieving 100% packet loss for malicious scans.  Integrated Snort IDS to monitor active penetration attempts and provide real-time intrusion alerts to security administrators.  Secured internal web services by transitioning to Apache HTTPS/TLSv1.3 and deploying a ModSecurity Web Application Firewall (WAF) to intercept and block OWASP vulnerabilities, such as SQL Injection attempts.  Enforced advanced Layer 2 access controls using Switch Port Security and VLAN Access Control Lists (VACLs).  Implemented secure wireless boundaries using WPA3-Enterprise with centralized RADIUS (IEEE 802.1X PEAP/TLS) authentication.
